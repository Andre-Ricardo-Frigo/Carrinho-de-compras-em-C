# Carrinho de Compras em C

Projeto de simulação de um carrinho de compras desenvolvido em linguagem C com interação via terminal.

---

## Descrição

A aplicação permite ao usuário selecionar produtos por código, definir quantidade e gerar uma nota fiscal com os valores calculados.

---

## Funcionalidades

- Listagem de produtos disponíveis  
- Seleção de produtos por código  
- Definição de quantidade  
- Cálculo de subtotal por item  
- Cálculo do total da compra  
- Impressão de nota fiscal formatada  

---

## Tecnologias

- Linguagem C  
- Estruturas (`struct`)  
- Manipulação de arrays  
- Entrada e saída padrão  

---

## Como executar

### Linux / macOS
```bash
gcc calculadora.c -o calculadora -lm && ./calculadora
```

### Windows (Prompt de Comando)
```bash
gcc calculadora.c -o calculadora.exe -lm && calculadora.exe
```
